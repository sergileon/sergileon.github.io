81       * This is a service definition, the service will be named "TimingFilter". The interface,
82       * RequestFilter, is used within the RequestHandler service pipeline, which is built from the
83       * RequestHandler service configuration. Tapestry IoC is responsible for passing in an
84       * appropriate Logger instance. Requests for static resources are handled at a higher level, so
85       * this filter will only be invoked for Tapestry related requests.
86       *
87       *
88       * Service builder methods are useful when the implementation is inline as an inner class
89       * (as here) or require some other kind of special initialization. In most cases,
90       * use the static bind() method instead.
91       *
92       *
93       * If this method was named "build", then the service id would be taken from the
94       * service interface and would be "RequestFilter".  Since Tapestry already defines
95       * a service named "RequestFilter" we use an explicit service id that we can reference
96       * inside the contribution method.
97       */
98      public RequestFilter buildTimingFilter(final Logger log)
99      {
100         return new RequestFilter()
101         {
102             public boolean service(Request request, Response response, RequestHandler handler)
103             throws IOException
104             {
105                 long startTime = System.currentTimeMillis();
106 
107                 try
108                 {
109                     // The responsibility of a filter is to invoke the corresponding method
110                     // in the handler. When you chain multiple filters together, each filter
111                     // received a handler that is a bridge to the next filter.
112 
113                     return handler.service(request, response);
114                 } finally
115                 {
116                     long elapsed = System.currentTimeMillis() - startTime;
117 
118                     log.info("Request time: {} ms", elapsed);
119                 }
120             }
121         };
122     }
